# 📺 Otium Streaming Service: Анализ и Спецификация Функционала

## 🚀 Обзор Проекта

Данный репозиторий содержит ключевые артефакты детального анализа и проектирования для **потокового видеосервиса Otium**. Фокус проекта — на моделировании взаимодействия между компонентами системы и создании формальной технической документации для внедрения конкретной функции.

Проект демонстрирует навыки глубокого анализа, применения нотаций UML для моделирования поведения системы и оформления требований в формате, готовом для передачи в разработку (Dev Ticket).

## 🎯 Ключевые Цели и Задачи

1.  **Моделирование Взаимодействия:** Разработка Диаграммы Последовательности (Sequence Diagram) для детальной визуализации процесса взаимодействия пользователя с сервисом при выборе контента.
2.  **Техническая Спецификация:** Формализация требований к новой функции "Фильтрация по эксклюзивному контенту" в виде **Тикета на разработку (Dev Ticket/Tech Spec)**, включая технические и функциональные детали.
3.  **Трассировка Требований:** Обеспечение четкой связи между аналитическим моделированием и финальным заданием для команды разработки.
