# 🏠 Stets Home: Проектирование Системы Управления "Умным Домом"

**Теги:** `System Analysis` `Smart Home` `DFD` `ERD` `Data Dictionary` `Figma` `QA Methodology` `Requirements Engineering`

## 🚀 Обзор Проекта

Данный проект представляет собой полный комплект артефактов системного и бизнес-анализа, разработанных для проектирования **Системы Управления "Умным Домом" Stets Home**. Система предназначена для мониторинга, автоматизации и удаленного управления бытовыми приборами и инженерными подсистемами (освещение, климат, безопасность) в жилых помещениях.

Проект демонстрирует навыки работы на фазе **Концептуального Проектирования, Детальной Спецификации Требований** и **Планирования Контроля Качества (QA)**.

## 🎯 Ключевые Цели и Задачи

1.  **Концептуальное Моделирование:** Разработка логических и контекстных моделей данных и процессов для определения границ системы.
2.  **Спецификация Данных:** Создание полного словаря данных для обеспечения единообразия и целостности информации.
3.  **Планирование QA:** Разработка программы и методики испытаний (ПМИ) для верификации и валидации разработанного ПО.
4.  **Прототипирование:** Создание ссылок на макеты (прототипы) интерфейса для визуализации пользовательского опыта.

## 📁 Структура проекта и Артефакты

Проект структурирован по фазам анализа и разработки. Все ключевые документы и диаграммы находятся в соответствующих папках.

| Папка | Ключевые Артефакты | Описание |
| :---: | :--- | :--- |
| **01_Requirements-SRS** | `Data_Dictionary_v2.pdf` | **Словарь Данных.** Полная спецификация всех сущностей, атрибутов и их характеристик, используемых в системе. |
| **02_Modeling-UML-DFD** | `DFD_Context_Level.jpg`, `DFD_Logical_Level.jpg` | **Диаграммы Потоков Данных (DFD).** Моделирование контекстного и логического уровня для отображения движения информации в системе. |
| **03_Data-Modeling-ERD** | `ER_Diagram_DFD_Context.jpg` | **Диаграмма "Сущность-Связь" (ERD).** Модель данных, отражающая структуру базы данных и связи между ключевыми сущностями системы "Умный Дом". |

## 🛠 Используемые Технологии и Нотации

* **Моделирование Процессов:** DFD (Data Flow Diagram)
* **Моделирование Данных:** ERD (Entity-Relationship Diagram)
* **Прототипирование и UI/UX:** Figma
* **Документация:** MS Word, PDF (Спецификации, Словари, ПМИ)

## ⚙️ Системные требования и инструкция по просмотру

Проект состоит из аналитической документации, схем и ссылок на макеты интерфейсов.
* **Документация:** Файлы в формате `.pdf` (Словарь данных, ПМИ, ссылки на макеты) можно открыть прямо в интерфейсе GitHub, в любом современном браузере или через Adobe Acrobat Reader.
* **Схемы:** Диаграммы в формате `.jpg` открываются стандартными средствами просмотра изображений.
* **Макеты:** Ссылки на прототипы, указанные в файле `Figma_Mockups_Link.pdf`, ведут на веб-версию Figma. Для просмотра макетов установка дополнительных программ не требуется, достаточно перейти по ссылке в браузере.

## 📈 Выводы по результатам проекта

В ходе проектирования системы Stets Home была успешно сформирована концептуальная архитектура умного дома. Использование DFD-диаграмм позволило проследить потоки данных от датчиков и пользовательских интерфейсов к центральному контроллеру. На основе этих потоков была спроектирована ER-модель и составлен исчерпывающий Словарь Данных, что исключает разночтения при дальнейшей разработке базы данных. Подготовленная Программа и Методика Испытаний (ПМИ) обеспечивает надежную базу для старта тестирования аппаратной и программной частей продукта сразу после их реализации.
