# 🏠 Stets Home: Проектирование Системы Управления "Умным Домом"

## 🚀 Обзор Проекта

Данный репозиторий представляет собой полный комплект артефактов системного и бизнес-анализа, разработанных для проектирования **Системы Управления "Умным Домом" Stets Home**. Система предназначена для мониторинга, автоматизации и удаленного управления бытовыми приборами и инженерными подсистемами (освещение, климат, безопасность) в жилых помещениях.

Проект демонстрирует навыки работы на фазе **Концептуального Проектирования, Детальной Спецификации Требований** и **Планирования Контроля Качества (QA)**.

## 🎯 Ключевые Цели и Задачи

1.  **Концептуальное Моделирование:** Разработка логических и контекстных моделей данных и процессов для определения границ системы.
2.  **Спецификация Данных:** Создание полного словаря данных для обеспечения единообразия и целостности информации.
3.  **Планирование QA:** Разработка программы и методики испытаний (ПМИ) для верификации и валидации разработанного ПО.
4.  **Прототипирование:** Создание ссылок на макеты (прототипы) интерфейса для визуализации пользовательского опыта.

## 📁 Структура Репозитория и Артефакты

Проект структурирован по фазам анализа и разработки. Все ключевые документы и диаграммы, подтверждающие мою экспертизу, находятся в соответствующих папках.

| Папка | Ключевые Артефакты | Описание |
| :---: | :--- | :--- |
| **01_Requirements-SRS** | `Data_Dictionary_v2.docx` | **Словарь Данных.** Полная спецификация всех сущностей, атрибутов и их характеристик, используемых в системе. |
| **02_Modeling-UML-DFD** | `DFD_Context_Level.jpg`, `DFD_Logical_Level.jpg` | **Диаграммы Потоков Данных (DFD).** Моделирование контекстного и логического уровня для отображения движения информации в системе. |
| **03_Data-Modeling-ERD** | `ER_Diagram_DFD_Context.jpg` | **Диаграмма "Сущность-Связь" (ERD).** Модель данных, отражающая структуру базы данных и связи между ключевыми сущностями системы "Умный Дом". |
