# 🛒 Ecommerce DB & Process Modeling: Проектирование Базы Данных и Бизнес-Процессов

## 🚀 Обзор Проекта

Данный репозиторий представляет собой коллекцию артефактов, демонстрирующих глубокую экспертизу в двух ключевых областях системного анализа: **Моделирование данных (Data Modeling)** и **Моделирование Бизнес-Процессов (BPMN)**.

Проект сосредоточен на разработке логических моделей для **Интернет-магазина**, включая структуру базы данных (ERD) и детализацию процессов взаимодействия с пользователем и операционной деятельности. Разнообразие представленных BPMN-диаграмм (от электронной коммерции до поликлиники) подчеркивает универсальность владения нотацией.

## 🎯 Ключевые Компетенции и Задачи

1.  **Моделирование Базы Данных (ERD):** Проектирование реляционной структуры, демонстрация нормализации и использование различных нотаций (Чена, "Воронья Лапка").
2.  **Моделирование Процессов (BPMN):** Детальное описание и визуализация бизнес-процессов (AS-IS и TO-BE), включая регистрацию пользователя и выполнение заказа.
3.  **Формализация Требований:** Создание точных, стандартизированных диаграмм, готовых к использованию разработчиками и бизнес-заказчиками.
